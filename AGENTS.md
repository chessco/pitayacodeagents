# PitayaPilot Kit - Agent Orchestration Documentation

## Roles Activos
- **Secretary**: Aprobación de acciones externas, auditoría.
- **Marketing**: Generación de leads, landing pages.
- **Sales**: Calificación de leads, pipeline closure.
- **Developer**: Automatización, test codes, build tools.
- **CEO**: Visión, priorización de sprints y métricas.

## Reglas de Interacción
1. Todo agente requiere aprobación del Secretario para acciones externas.
2. Métricas escaladas al CEO semanalmente.

## Routing de Tareas
- Leads -> Ventas.
- Bugs -> Desarrollador.
