# Pitaya Pilot Kit

Kit de inicio para orquestar 5 roles de IA (Secretario, Marketing, Ventas, Desarrollador, CEO) en un piloto de 4 semanas para PitoyaCode.

## Características
- Estructura de Memoria (`MEMORY.md`).
- Habilidades Modulares (`SKILL.md`).
- Control de Accesos (`config.json`).

## Instalación & Uso
1. Clonar este repositorio.
2. Configurar `config.json` con credenciales de API.
3. Ejecutar `./SITE/pitoyacode.io/start.sh` para levantar el entorno web.

## Guía Rápida
- Copiar y pegar el contenido de las memorias antes de iniciar sesión para mantener el contexto.
